Utility

search

Quick ADS paper search that prints results directly to the terminal. Useful for finding bibcodes before running a workflow or testing a query before adding a watch.

Usage

shell
almagest search "dark matter substructure year:2023-2025"
almagest search "author:Schaye,J bibstem:MNRAS" --sort "date desc"
almagest search "doctype:review abs:AGN feedback" --limit 20

Options

OptionDefaultDescription
--limit10Maximum number of results to return.
--sortcitation_count descSort order. Common values: citation_count desc, date desc.

Each result shows: bibcode, first author, year, title, and citation count. Use almagest show <bibcode> to fetch the full abstract for any result.

ADS query syntax — search supports the full ADS Solr syntax. See the ADS query syntax reference for a complete list of operators.
Utility

show

Fetch and display the full metadata and abstract for a single ADS bibcode. Useful for quickly reading a paper's content before deciding whether to include it in a workflow.

Usage

shell
almagest show 1997ApJ...490..493N
almagest show 2023Natur.616..266L

Output

Displays: title, authors, year, journal, citation count, bibcode, DOI, arXiv link, and the full abstract.

Utility

config-check

Verifies that all required environment variables are present and reachable before running any workflow. Run this after setting up your .env file to confirm everything is configured correctly.

Usage

shell
almagest config-check

Example output

terminal
╭─ Configuration Check ─╮
  ✓ ADS_API_TOKEN: abc123...
  ✓ LLM_PROVIDER: claude
  ✓ ANTHROPIC_API_KEY: sk-ant-...
    model: claude-sonnet-4-6
    output dir: ./outputs

All checks passed.

If a check fails, the command prints the missing variable and exits with a non-zero status. Copy .env.example to .env and fill in the missing values, then re-run.
